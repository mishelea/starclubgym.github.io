Nuestras Máquinas

¿Qué es el Índice de Masa Corporal (IMC)?

El Índice de Masa Corporal (IMC) es una medida que relaciona el peso y la estatura para determinar si una persona tiene un peso saludable. Se utiliza como una referencia para evaluar el riesgo de enfermedades relacionadas con el peso.
